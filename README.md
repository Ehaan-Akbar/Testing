# Codex vs Claude Code

This repository contains the source code, examples, and supporting materials used in the accompanying article comparing Codex and Claude Code.

The goal of the comparison was to evaluate how each tool performs across a programming. All code included here was generated, modified, or reviewed by AI as part of that evaluation process.

This repository is not intended to be a benchmark suite. It simply provides the code for the outputs referenced throughout the article so readers can inspect the implementations, reproduce the examples, and draw their own conclusions.

For details on the methodology, prompts, observations, and results, refer to the article.
